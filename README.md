# Machine Learning — Homework Assignments

**Course:** Machine Learning (Aprendizagem) — IST, University of Lisbon
**Year:** 2023/2024

## Description

Four homework assignments covering core machine learning topics:

| HW | Topics |
|----|--------|
| **HW1** | Data exploration, feature analysis, baseline classifiers |
| **HW2** | Decision trees, model evaluation, bias-variance tradeoff |
| **HW3** | Ensemble methods, SVMs, wine quality prediction |
| **HW4** | Neural networks, clustering, dimensionality reduction |

Each assignment includes both **pen-and-paper** theoretical exercises and **programming** notebooks.

## Technologies

- **Language:** Python
- **Libraries:** Scikit-Learn, NumPy, Pandas, Matplotlib
- **Dataset:** UCI Column Diagnosis, Wine Quality

## Project Structure

```
HW1/    # Data exploration and baseline classifiers
HW2/    # Decision trees and model evaluation
HW3/    # Ensemble methods and SVMs
HW4/    # Neural networks and clustering
```
